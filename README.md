## Features

### User Features
- User can **register** (if new user)
- User can **login**
- User can **view lost items**
- User can **view found items**
- User can **search items**
- User can **mark item as returned**

### Admin Features
- Admin can **add lost items**
- Admin can **add found items**
- Admin can **view all lost items**
- Admin can **view all found items**
- Admin can **search items**
- Admin can **delete items**
- Admin can **mark items as returned**

## User Roles
- **User**
- **Admin**

Each role has different permissions inside the  system.
